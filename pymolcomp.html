goChem

A library for computational chemistry.

For convenience, we provide binaries of our PyMOL plugins for a number of OS/architectures. Find the sources here.

goQM, a plugin for quick quantum chemistry calculations on protein PyMOL selections.

  • Linux, 64 bits.
  • Linux, 32 bits (x86).
  • MacOS, 64 bits.

goRama, a plugin for flexible Ramachandran plot creation from PyMOL selections.

  • Linux, 64 bits.
  • Linux, 32 bits (x86).
  • MacOS, 64 bits.

If you have ideas for PyMOL plugins that you would like implemented, other suggestions or need with the binaries, please contact the developer.
